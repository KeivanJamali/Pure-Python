\section{Introduction}

In this part, you will extend your optimization knowledge to multi-variable problems using the Gradient Descent algorithm.

This problem represents a simplified version of flow distribution across parallel routes in a transportation network.

%------------------------------------------------------
\subsection{Flow Optimization on Parallel Links}

Consider 4 parallel links connecting the same origin and destination. The total demand is:

$$ D $$

Flows must satisfy:

$$ x_1 + x_2 + x_3 + x_4 = D $$

Each link follows the BPR travel time function:

$$ T_i(x_i) = T_i^0 \left(1 + B \left(\frac{x_i}{C}\right)^4 \right) $$

The total system cost is:

$$ Z(x_1, x_2, x_3, x_4) = \sum_{i=1}^{4} x_i \cdot T_i(x_i) $$

\textbf{Instructions:}
\begin{enumerate}
    \item Implement \textbf{Gradient Descent}
    \item Minimize \( Z(x_1, x_2, x_3, x_4) \)
    \item Enforce the constraint:
    $$ x_1 + x_2 + x_3 + x_4 = D $$
\end{enumerate}

\textbf{Hint:}
You may:
\begin{itemize}
    \item Use a penalty function:
    $$ Z'(x) = Z(x) + \mu (x_1 + x_2 + x_3 + x_4 - D)^2 $$
\end{itemize}
