\section*{Part 7: SUMO Simulation with the Same Configuration}

\textbf{Goal:}
\begin{enumerate}
\item Explore libsumo.
\item Reproduce the Python-based traffic simulation in the SUMO traffic simulator.
\item Use Python to generate all inputs and configuration files.
\end{enumerate}

\subsection*{Tasks}
\begin{itemize}
    \item Learn the basics of SUMO: network definition, route files, and configuration files.
    \item Convert the previously defined network (nodes, edges, intersections) into SUMO input files.
    \item Generate vehicle demand using the same logic as in the Python simulation.
    \item Run the simulation in SUMO and visualize the traffic flow.
    \item Compare the SUMO results with the custom Python simulation.
\end{itemize}

\subsection*{Output}
\begin{enumerate}
\item SUMO input and configuration files (\texttt{.net.xml}, \texttt{.rou.xml}, \texttt{.sumocfg}).
\item Comparison report.
\item Present the code and run the simulation.
\end{enumerate}

\subsection*{Time}
2 weeks (1404-10-16)

\subsection*{Grade Weight}
25\% (.net.xml: 5\%, .rou.xml: 5\%, runs without errors: 5\%, correct outputs: 10\%)
