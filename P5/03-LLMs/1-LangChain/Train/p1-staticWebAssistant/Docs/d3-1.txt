## Project Overview  
This project was developed for a **Systems Engineering course**, focusing on creating a Python library that simplifies and enhances the transparency of the **Simplex method**.  

---

## Project Steps  
1. **Performing the Simplex algorithm** – the magic starts here!  
2. **Prepping the two-phase Simplex method** – ensuring readiness for any scenario.  
3. **Implementing sensitivity analysis** – understanding how changes impact solutions.  
4. **Using duality for optimization** – duality reveals the secret sauce for efficiency!
